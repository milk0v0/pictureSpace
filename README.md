## 技术栈
+ node
+ koa(koa-body、koa-jwt、koa-router、koa-static-cache)
+ jsonwebtoken
+ koa-server-http-proxy
+ mysql2

## 启动
+ npm run 1 -> node app.js -> 提供服务服务器
+ npm run 2 -> node staticServer.js -> 提供静态资源服务器

## 页面
+ 注册页 register.html
+ 登录页 login.html
+ 相册页 index.html
